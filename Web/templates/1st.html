What is Magnet

Magnet aims to be a social sentiment analysis tool.

It classifies tweets (from a Twitter query) or plain text as positive or negative, if sentiment is found. Otherwise, the text is mapped as objective.

How Magnet works

Magnet uses a Naive Bayes classifier to evaluate whose class fits better given a set of features exctracted by the submitted text (from query result or plain text.)

How tweet can be considered objective?

A tweet can be considered objective if it does not contain a polar word. Polar words are special words which expresses (or can express) a sentiment.
